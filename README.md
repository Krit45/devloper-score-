## 🌟 Overview

DevScore is an AI-powered platform that analyzes GitHub profiles and
provides developer insights, scoring, and improvement suggestions.

------------------------------------------------------------------------

## ✨ Features

-   🔍 GitHub Profile Analysis\
-   📊 DevScore Calculation\
-   📈 Visual Charts\
-   🤖 AI Suggestions\
-   ⚡ Fast UI

------------------------------------------------------------------------

## 🛠️ Tech Stack

-   React, Vite, Tailwind CSS\
-   Node.js, Express\
-   Firebase, Google GenAI

------------------------------------------------------------------------

## ⚡ Getting Started

``` bash
1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
```

------------------------------------------------------------------------

## 🔑 Environment Variables

    GEMINI_API_KEY=your_api_key
    GITHUB_TOKEN=your_token
